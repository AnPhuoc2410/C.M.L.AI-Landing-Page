# Nếu Mác gặp ChatGPT 🤖🎓

Trang web tương tác khám phá cuộc gặp gỡ triết học giữa Karl Marx và trí tuệ nhân tạo ChatGPT. Đây là một dự án học thuật độc đáo kết hợp giữa triết học Mác-xít cổ điển và công nghệ AI hiện đại.

## Nội dung chính

### 4 Điểm nhấn nghiên cứu:

1. **AI và lực lượng sản xuất**
   - AI như một bước nhảy vọt trong lực lượng sản xuất
   - Tác động đến quan hệ sản xuất hiện tại

2. **Lao động và giá trị thặng dư**
   - Máy móc thông minh thay thế con người
   - Ai sở hữu AI sẽ chiếm giá trị thặng dư?

3. **Con người trong kỷ nguyên AI**
   - AI có làm mất đi bản chất sáng tạo của con người?
   - Hay giải phóng con người khỏi lao động nặng nhọc?

4. **Đạo đức và triết học**
   - Áp dụng phép biện chứng duy vật
   - AI như "phủ định của phủ định" trong tiến hóa xã hội

## Tính năng

✨ **Giao diện tương tác**: Sử dụng GSAP cho animations mượt mà  
🎨 **Thiết kế học thuật**: Color palette kết hợp giữa màu sắc triết học và công nghệ  
📱 **Responsive**: Tương thích với mọi thiết bị  
🎭 **Storytelling**: Kể chuyện qua visual design và typography  

## Công nghệ sử dụng

- **Frontend**: React + Vite
- **Animations**: GSAP (GreenSock)
- **Styling**: Tailwind CSS
- **Typography**: Antonio + Crimson Text fonts

## Scripts

```bash
npm install
npm run dev
npm run build
```
